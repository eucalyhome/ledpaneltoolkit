# ledpaneltoolkit
matrix ledpanel toolkit for python / raspberry pi / rpi-rgb-led-matrix

# 概要
*マトリックスLEDパネルの、簡易制御
*Web I/F
*インフォメーションパネルとして使える各種ツールキットを提供

# 使用資材
*Raspberry pi model3B / 3B+
zero系は処理速度が足りません

*MATRIX LED I/F
adafruit HAT もしくは 自作系

*赤外線I/F及びリモコン
Apple Remoteを想定して制作しています

# セットアップ
*SETUP.md を参照して下さい
